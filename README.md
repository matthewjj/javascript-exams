## 2 Indicies Test

Find 2 indicies from array that sum to target a give number:

### Usage

```
const indiciesService = require('../library/array-2-indicies.js');

const nums = [2, 7, 11, 15];
let target = 9;

console.log(
    indiciesService.map(nums, target)
);
```
